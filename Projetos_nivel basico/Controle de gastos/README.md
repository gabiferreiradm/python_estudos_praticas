# Controle de notas em Python

Projeto básico em Python para registro, validação e análise de notas
de alunos informadas via terminal.

## Funcionalidades
- Registro de notas válidas (0 a 10)
- Validação de entradas inválidas
- Cálculo automático da média
- Identificação de maior e menor nota
- Resultado final de aprovação ou reprovação

## Tecnologias
- Python 3

## Como executar
1. Certifique-se de ter o Python 3 instalado.
2. No terminal, navegue até a pasta do projeto.
3. Execute o comando:

```bash
python controle_notas.py
